# Flutter Plant App

## Giriş / Intro ##
* Flutter Plant App, Flutter ile geliştirilen basit bir mobil uygulamadır. Uygulama, kullanıcıların Firebase ile kayıt olup giriş yapabilmelerini, misafir girişi yapabilmelerini ve Firebase'de çeşitli türdeki bitkilerin fotoğrafları, isimleri ve resimleri gibi detayları depolayabilmelerini sağlar. Uygulama, Türkçe ve İngilizce olmak üzere iki farklı dil desteği sunar.
* Flutter Plant App is a simple mobile application developed with Flutter. The application allows users to register and log in via Firebase, offering a guest login option as well. Users can store various details about plants, such as photos, names, and images, in Firebase. The app provides language support in both Turkish and English.

## Ön Koşullar / Prerequisites ##

Bu projeyi çalıştırmak için aşağıdaki araçlara ihtiyacınız olacak:

- Flutter SDK
- Android Studio veya Xcode
- Firebase hesabı ve projesi

Bu projeyi kullanmak için aşağıdaki izinlere ihtiyacınız olacak:

- İnternet erişimi
- Galeri erişimi
- Kamera erişimi

You'll need the following tools to run this project:

- Flutter SDK
- Android Studio or Xcode
- Firebase account and project

To use this project, you'll need the following permissions:

- Internet access
- Gallery access
- Camera access

## Özellikler ve İşlevler / Features and Functions ##

Uygulamanın sunduğu özellikler ve işlevler şunlardır:

- [x] Kullanıcılar, e-posta ve şifre ile kayıt olabilir veya giriş yapabilirler.
- [x] Kullanıcılar, misafir olarak da uygulamayı kullanabilirler.
- [x] Kullanıcılar, bir öge oluşturabilirler. Bir öge için bir isim, bir tür ve bir resim seçmelidirler. Resim, galeriden veya kameradan alınabilir.
- [x] Kullanıcılar, oluşturdukları ögeleri Firebase'de depolayabilirler.
- [x] Kullanıcılar, ana ekranda depoladıkları ögeleri bir liste olarak görebilirler.
- [x] Kullanıcılar, uygulamanın dilini Türkçe veya İngilizce olarak değiştirebilirler.
- [x] Kullanıcılar, uygulama hakkında temel bilgileri ve nasıl kullanılacağını görebilirler.

Here are the features and functions provided by the application:

- [x] Users can register or log in with email and password.
- [x] Users can also use the application as guests.
- [x] Users can create an item. They need to provide a name, a type, and an image for an item. The image can be taken from the gallery or the camera.
- [x] Users can store the created items in Firebase.
- [x] Users can view the items they stored as a list on the home screen.
- [x] Users can switch the application's language between Turkish and English.
- [x] Users can access basic information about the application and how to use it.
